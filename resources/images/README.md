# Google Material Symbols Outlined — 48 px

This package contains locally stored SVG icons for OpcUaManager.

Style:
- Material Symbols Outlined
- Weight: 400 (Regular)
- Fill: 0
- Grade: 0
- Optical size: 48 px

The files were taken from the official Google material-design-icons repository
and renamed locally to stable, concise filenames. Their SVG path data was not
modified.

Suggested QML usage:

    icon.source: "qrc:/images/svg/search.svg"
    icon.width: 24
    icon.height: 24

The SVG source is vector-based and may be rendered at other sizes. The 48 px
variant was selected as the source master for this project.
